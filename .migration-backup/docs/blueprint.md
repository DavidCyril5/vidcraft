# **App Name**: VidCraft AI

## Core Features:

- Video Generation Interface: An intuitive user interface for inputting text prompts and initiating video generation using the available AI models (Veo-3.1, Grok Video, Seedance 2.0).
- AI Model & Ratio Selection: Users can easily select their preferred AI video generation model and specify the desired aspect ratio (16:9, 9:16).
- Image-to-Video Option: Capability to generate videos from provided image URLs, enhancing creative possibilities.
- Prompt Enhancement Tool: An AI tool that assists users in crafting more detailed and effective video prompts from brief ideas or keywords.
- Generated Video Playback: Display the completed AI-generated video in a responsive player, allowing users to view their creation.
- Generation Status & Error Display: Clear indicators for the video generation progress, completion, or any API-related errors.

## Style Guidelines:

- Primary brand color: A bright, electric violet (#C999FF) symbolizing creativity and digital innovation, ensuring high visibility on darker backgrounds.
- Background color: A very dark, subtle purplish-grey (#19151A) to establish a sophisticated and futuristic aesthetic.
- Accent color: A lighter, softened blue (#8AB2F0) for secondary interactive elements, providing a pleasant contrast and complementing the primary violet.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful appearance.
- Body font: 'Inter' (sans-serif) for clear, highly readable text in all content areas.
- Minimalist and modern line icons that abstractly represent concepts like 'generate,' 'video,' 'upload,' and 'settings,' maintaining a clean interface.
- A responsive, centered layout focusing on the generation process with prominent input fields and a large area for video playback. Adaptive to both desktop and mobile views.
- Subtle, fluid transitions for state changes and engaging loading animations during video processing to provide user feedback.
